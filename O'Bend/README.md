# O'Bend

CEP-плагін для Adobe Illustrator. Математично згинає, скручує в кільця і робить 3D-спіралі/гелікси з векторних об'єктів — з підтримкою груп та compound paths.

## Як працює

1. Виберіть один або кілька векторних об'єктів (path / group / compound path)
2. Налаштуйте параметри
3. Натисніть `APPLY`. Скасування — `Ctrl+Z`

**Axis** — навколо якої осі гнути:
- **HORIZONTAL** — горизонтальна (rotate 0°)
- **VERTICAL** — вертикальна (rotate 90°)
- **CUSTOM** — довільний кут

**Bend параметри:**
- **Subdivisions (0-7)** — рівень підрозбиття. Прямі лінії треба ділити, інакше вони не зігнуться. Level 0 = без змін, Level 5-6 = плавно. Рівень 7 = 128 сегментів на лінію
- **Bend Angle (-1080° до 1080°)** — кут згинання. 360° = повне кільце, 180° = півколо
- **Limit (%)** — відсоток об'єкта, який підпадає під згинання
- **Center (%)** — точка центру bend всередині об'єкта (0% = початок, 50% = середина)
- **Offset (Radius)** — зміщення від центру обертання, створює "діру" всередині кільця
- **Direction** — Normal або Reverse

**Helix параметри:**
- **Radial Expand** — нарощування радіуса з кожним обертом (рулон стрічки)
- **Axis Shift (Z)** — зсув уздовж осі з кожним обертом (3D-пружина)

## Поради

- На складних об'єктах починайте з Subdivisions 2-3
- Розрізні петлі на внутрішньому радіусі helix — це не баг, це 2D-проекція 3D трохоїди. Збільште Axis Shift або зменшіть товщину оригіналу
- Reset-кнопки на кожному слайдері повертають конкретний параметр до дефолту, а кнопка RESET внизу — всі одразу
