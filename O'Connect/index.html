Curve

Tension shapes the curvature (0 = straight, 1 = round). Angle threshold controls the width of horizontal Left/Right sectors; larger means more side connections.

Stroke

Hub = front-most selected object. Other selected objects become children. Each child gets a smooth bezier curve to the hub edge facing it.

Connecting to Illustrator...

Bring the Hub object to the front (Object > Arrange > Bring to Front), then select Hub + children.
