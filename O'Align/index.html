Direction

AUTO snaps to whichever axis is closer (≤45° = horizontal, otherwise vertical). HORIZONTAL/VERTICAL force a specific result.

Pivot

Pick 2 anchor points (Direct Selection / White Arrow) or select a guide line, then run ALIGN. The parent object rotates so the two points sit on a horizontal or vertical axis.
