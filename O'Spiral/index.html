Loop Mode

TOTAL spreads loops across the whole spiral. PER SEGMENT multiplies loops by the number of segments between circles.

Loops

Randomness

Adds per-loop expansion noise. Pattern re-rolls only when this slider moves; other parameters reuse the same noise.

Density

Density: anchors per loop. Tension: bezier handle length as % of tangent. 33 ≈ default Catmull-Rom; 0 = corner; negative = inverted handles (cusps).

Direction

Select 2 or more circles, then press SPIRAL.

Connecting to Illustrator...

Circles are sorted top-down then left-to-right (10pt vertical tolerance). Spiral inherits stroke colour from the first circle (or black if not stroked).
