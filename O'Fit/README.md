# O'Fit

CEP-плагін для Adobe Illustrator. Підганяє виділені об'єкти під розмір активного артборду — з вибором режиму, відступів та позиції.

## Як працює

1. Виберіть один або кілька об'єктів
2. Налаштуйте режим, padding і align
3. Натисніть `FIT TO ARTBOARD`

**Fit Mode:**
- **CONTAIN** — пропорційно вмістити цілком всередину (як в оригінальному скрипті)
- **COVER** — пропорційно заповнити, з можливим виходом за межі
- **STRETCH** — без збереження пропорцій, точно по розмірах артборду

**Padding** — горизонтальний/вертикальний відступ від країв артборду в пікселях (вирізається з обох сторін)

**Align** — 3×3 сітка позицій. Для CONTAIN визначає кут / сторону прив'язки. Для COVER — звідки буде crop.

Кожен об'єкт виділення обробляється окремо — можна підігнати кілька варіацій одним кліком.
