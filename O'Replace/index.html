Geometry

Color (Pure Replace)

Each target is replaced 1:1 — clipboard fits the target's bounds and inherits the target's fill color. No brightness modulation.

Light (Brightness-modulated)

Each replacement is scaled by the target's perceived brightness — white targets stay full, dark ones shrink. Color is inherited.

Copy a vector object (Ctrl+C), select target paths/groups, then press REPLACE.

Connecting to Illustrator...

RESET restores the LIGHT tab defaults. APPLY commits the preview (and removes originals if checked).
