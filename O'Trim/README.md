# O'Trim

CEP-плагін для Adobe Illustrator. Розрізає виділений об'єкт на сітку клітинок, кожна клітинка — clipping mask + копія оригіналу. Кожна клітинка масштабується степенево залежно від відстані до якоря (anchor cell), потім всі клітинки збираються назад зі заданим Gap.

## Як працює

1. **Mode** обирає форму сітки:
   - `COLS` — горизонтальний розклад, X колонок × 1 рядок
   - `ROWS` — вертикальний, 1 колонка × Y рядків
   - `GRID` — повна сітка X × Y
2. **Anchor** — клітинка, від якої міряється відстань для масштабування (Center або кутова).
3. **Scale (%)** — коефіцієнт. Кожна клітинка отримує `Scale^distance` від якоря: при 80% сусіди по 80%, через одного 64% і т.д.
4. **Proportional Scale** — чи масштабувати обидві осі однаково, чи тільки вісь, що відповідає Mode.
5. **Gap** — відступ між клітинками після збирання (в одиницях документу).

Оригінальний об'єкт спочатку групується як страхувальна копія, потім повністю замінюється результатом.
